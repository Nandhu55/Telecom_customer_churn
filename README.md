# 📱 Telecom Customer Churn Prediction

A Machine Learning web application built using Streamlit to predict whether a telecom customer is likely to churn or not.

## 🚀 Live Demo

[Telecom Customer Churn App](https://telecomcustomerchurn-1.streamlit.app/?utm_source=chatgpt.com)

---

## 📌 Features

- Predict customer churn instantly
- Interactive Streamlit UI
- Logistic Regression Machine Learning model
- User-friendly input form
- Real-time prediction results

---

## 🛠️ Technologies Used

- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- Pickle
